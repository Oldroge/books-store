### Como executar o projeto:
faça o clone do projeto na sua máquina local:
```bash
git clone git@github.com:Oldroge/books-store.git
```

Navega até a pasta do projeto:
```bash
cd books-store
```

Instale as dependências:
```bash
npm install
```

Para executar o projeto digite:
```bash
npm start
```
OU
```bash
ng serve
```
